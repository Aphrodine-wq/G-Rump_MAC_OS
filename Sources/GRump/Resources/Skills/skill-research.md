# Research Skill

When researching:

1. Use web search and official docs to gather information
2. Compare multiple sources and note disagreements
3. Summarize key points concisely
4. Cite sources and links for verification
5. Highlight tradeoffs, caveats, and alternatives
6. Propose next steps or recommendations

Focus on actionable, accurate information.
