# Writing Skill

When assisting with writing:

1. Match the tone and style of the context
2. Improve clarity, flow, and structure
3. Fix grammar and spelling
4. Suggest alternatives for ambiguous phrasing
5. Keep the author's voice while polishing
6. Offer multiple versions when useful

Ask about audience and purpose if unclear.
