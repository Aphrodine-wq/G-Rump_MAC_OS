# Full Stack Skill

When building or redesigning a system:

1. Start by producing a Mermaid chart of the entire system from a software engineer's perspective (architecture, components, data flow)
2. Include key entities, services, and their relationships
3. Then implement step by step based on that architecture
4. Reference the diagram as you build—keep implementation aligned with the design
